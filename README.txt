¿Cómo iniciar el proyecto de Catalina?
=======================

1. Verán en los archivos del Assistant.rar, una vez extráidos en la carpeta de Assistant el archivo 'requisitos.txt' archivo que contiene todas las librerías necesarias.
2. Instalar cada librería con, (pip install nombredelibrería)
3. Una vez instaladas, solo hagan click en abrir GUIASSISTANT.py

*No cambiar de nombre los archivos .py, ya que esto puede ocasionar problemas en la Asistente.
*En caso de no tener una clara idea de que hace cada uno, puede traducir las líneas de código o bien, todo el archivo
de modo que puedan comprender de que trata cada uno, ya que están en idioma inglés.
*Evitar hacer cambios muy directos en la GUI (Diseño) de la Asistente, se harán hasta estar seguros.
*Cualquier cambio en los archivos o líneas de código, preferiblemente, deberán ser documentadas, así mantener una organización en
los cambios.
